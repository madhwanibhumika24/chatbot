# Intelligent Assistant

An AI-powered assistant built using LangChain, Google Gemini and Streamlit.

## Features

- LangChain
- Google Gemini
- Streamlit Chat Interface
- Chat History
- Structured Responses

## Tech Stack

- Python
- LangChain
- Google Gemini
- Streamlit

## Run

```bash
pip install -r requirements.txt
streamlit run app.py
